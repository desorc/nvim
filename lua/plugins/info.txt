* obsidis.nvim:
  A Neovim plugin for writing and navigating Obsidian vaults
* orgmode:
  Use your favourite package manager
* copilot.lua
  This plugin is the pure lua replacement for github/copilot.vim. Copilot uses OpenAI Codex to suggest code and entire functions in real-time right from your editor. Trained on billions of lines of public code, GitHub Copilot turns natural language prompts including comments and method names into coding suggestions across dozens of languages
* copilot-cmp
  This repository transforms https://github.com/zbirenbaum/copilot.lua into a cmp source.
  Copilot suggestions will automatically be loaded into your cmp menu as snippets and display their full contents when a copilot suggestion is hovered
* vim-dadbod-completion:
  Database auto completion powered by vim-dadbod

=== cmp ===
	A completion engine plugin for neovim written in Lua. Completion sources are installed from external repositories and "sourced"

=== delimitMate ===
	automatic closing of quotes, parenthesis, brackets, etc., besides some other related features that should make your time in insert mode a little bit easier, like syntax awareness (will not insert the closing delimiter in comments and other configurable regions), and expansions (off by default), and some more

=== gruvbox-material ===

=== hop ===
